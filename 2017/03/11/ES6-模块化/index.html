export default 命令

为了快速上手，用户往往没有时间去看文档来学习需要导入的变量名，因此需要通过export default指令来输出

本质上是将输出的变量赋值给了一个叫做default的变量，在导入时，default可以被任意命名

export 与 import 复合写法

export { foo, bar } from 'my_module';

等价于

import { foo, bar } from 'my_module';
export { foo, bar };
